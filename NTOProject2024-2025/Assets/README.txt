============== Для работы с сохранениями =======================
Чтобы очистить всю информацию о персонажах:
- На сцене Bootstrap нажмите галочку inEditor на объекте EntryPoint
- Начните игру, но ничего не делаете
- Завершаете Run 
- Убираете галочку

Со следующего сеанса игра будет работать должным образом

Если выдает какую то ошибку, связанную с запросом сервера, пишем чувачку @Philtwe
