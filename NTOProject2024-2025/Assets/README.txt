============== Для работы с сохранениями =======================
Чтобы очистить всю информацию о персонажах:
- На сцене MainMenu нажмите галочку inEditor на объекте EntryPointMainMenu
- Начните игру, но ничего не делаете
- Завершаете Run 
- Убираете галочку

Со следующего сеанса игра будет работать должным образом
